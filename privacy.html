Privacy Policy

This extension does not collect, store, or transmit any personal data.

All functionality runs locally in the user's browser.

데이터 수집 및 저장

Dot Cat Walker는 다음의 설정 데이터를 사용자의 브라우저에 로컬로 저장합니다:

  • 확장 프로그램 활성화 상태
  • 선택된 고양이 목록
  • 발자국 효과 ON/OFF 설정
  • 커서 반응 모드 설정
  • 각 고양이의 크기 설정

데이터 보안

모든 설정 데이터는 사용자의 로컬 브라우저(chrome.storage.local)에만 저장되며, 외부 서버로 전송되지 않습니다. 개인 정보나 민감한 데이터는 수집하지 않습니다.

---

Data Collection and Storage

Dot Cat Walker stores the following settings data locally in your browser:

  • Extension enable/disable state
  • Selected cats list
  • Paw prints effect ON/OFF setting
  • Cursor reaction mode setting
  • Individual cat size settings

Data Security

All settings data is stored only in your local browser (chrome.storage.local) and is never transmitted to external servers. We do not collect personal or sensitive information.
