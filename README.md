# Dot Cat Walker

모든 웹 페이지 위를 돌아다니는 픽셀 고양이 Chrome 확장 프로그램입니다.

## ✨ 주요 기능

### 🐱 다중 고양이 지원
- **여러 마리 고양이 동시 활성화**: 3종류의 고양이를 원하는 조합으로 선택 가능
- 각 고양이는 독립적으로 움직이며 고유한 상태를 가짐

### 📏 개별 크기 조절
- **고양이별 크기 설정**: 각 고양이마다 1x ~ 4x 크기 선택 가능
- 팝업에서 고양이 이미지 아래 버튼으로 간편하게 조절

### 🎭 상태 시스템
- **IDLE**: 멈춰서 쉬는 상태
- **WALK**: 화면을 자유롭게 걸어다니는 상태
- **SLEEP**: 잠자는 상태
- **ONENTER**: 마우스 호버 시 반응
- **DRAG**: 드래그로 고양이 이동 가능

### 🖱️ 커서 반응 모드
- **없음**: 일반적인 랜덤 이동
- **호기심 (쫓아감)**: 마우스에 가까이 가면 쫓아옴 (1.5배 속도)
- **수줍음 (도망감)**: 마우스에 가까이 가면 도망감 (1.5배 속도)
- 자고 있거나 멈춰있는 고양이도 마우스에 반응하여 깨어남!

### 🐾 발자국 효과
- 고양이가 걷는 곳에 발자국 표시 (토글 가능)
- 고양이별로 다른 색상의 발자국
  - Cat1: 기본 색상
  - Cat2: 진한 회색
  - Cat3: 밝은 회색
- 2초 후 자동으로 사라짐

## 🎮 사용법

### 설치

1. 이 저장소를 클론합니다.
   ```bash
   git clone https://github.com/QWER-ASDF-1234/cat-walker.git
   cd cat-walker
   ```

2. 의존성을 설치하고 빌드합니다.
   ```bash
   npm install
   npm run build
   ```

3. Chrome에서 `chrome://extensions`로 이동합니다.

4. **개발자 모드**를 활성화합니다.

5. **압축해제된 확장 프로그램을 로드합니다**를 클릭하고 프로젝트 폴더를 선택합니다.

### 기본 사용

1. **확장 아이콘 클릭** - 팝업 메뉴가 열립니다

2. **ON/OFF 토글** - 고양이 활성화/비활성화

3. **고양이 선택**
   - 3종류의 고양이 중 원하는 고양이 클릭
   - 여러 마리 동시 선택 가능
   - 빠른 액션: "모두 선택" / "모두 해제" 버튼

4. **크기 조절**
   - 각 고양이 이미지 아래 1, 2, 3, 4 버튼 클릭
   - 개별적으로 크기 설정 가능

5. **추가 기능**
   - **발자국 효과**: 토글로 켜고 끌 수 있음
   - **커서 반응**: 없음 / 호기심 / 수줍음 선택

### 상호작용

- **마우스 호버**: 고양이에 마우스를 올리면 특별한 애니메이션
- **드래그**: 고양이를 클릭하고 드래그하여 원하는 위치로 이동
- **커서 반응**: 마우스가 가까이 가면 쫓아오거나 도망감 (200px 범위)

## 🔧 기술 스택

- **Chrome Extension Manifest V3**
- **TypeScript**
- **Sprite Animation** (32x32px @ 4x scale)
- **Chrome Storage API**
- **RequestAnimationFrame** - 60fps 애니메이션
